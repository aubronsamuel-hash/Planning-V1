🗝️ Glossaire express

Projet
Un spectacle ou une production (ex: Peter Pan, Garou Tournée 2026)
Représentation
Une occurrence unique d'un projet : un jour, un lieu, des horaires
Affectation
L'assignation d'un collaborateur à une représentation pour un poste
Poste requis
Un besoin en personnel sur un projet (ex : "2 Éclairagistes")
Équipe
Sous-groupe de postes/collaborateurs sous un Chef de poste
Cachet
Rémunération forfaitaire par représentation (intermittents)
DPAE
Déclaration Préalable À l'Embauche — obligatoire avant 1ère prestation
Compte GHOST
Collaborateur invité sans mot de passe — accès via magic links uniquement
Magic link
Lien email temporaire donnant accès sans connexion (UUID, usage unique)
Confirmation atomique
Chaque date confirmée/refusée indépendamment, effet immédiat
Get-in / Show / Get-out
Créneaux horaires d'une représentation (installation → spectacle → démontage)
isReadOnly
Mode lecture seule si trial expiré ou paiement échoué — données conservées

📌 Décisions clés en un coup d'œil

✅Abonnement mensuel par organisation (pas par utilisateur)
✅Confirmation d'affectation atomique — date par date, effet immédiat dans la grille
✅Lazy Auth — compte fantôme (GHOST) → activation volontaire (ACTIVE)
✅Salle variable par représentation (pas de salle fixe par projet / tournée)
✅Export iCal (abonnement dynamique) + CSV paie (SAGE / Cegid)
✅RGPD : anonymisation automatique après 3 ans d'inactivité (cron mensuel)
✅Créneaux horaires : Get-in / Warmup / Show / Get-out définis par représentation
✅9 types de projets (Théâtre, Concert, Cirque, Opéra, Danse, Maintenance…)
✅3 niveaux de rôles : plateforme (User.role) · organisation (OrganizationMembership) · équipe (EquipeMembre)
✅Plans : FREE (0€/3 collabs) · PRO (49€/20 collabs) · ENTERPRISE (149€/illimité) · Trial 14j
✅SSE (Server-Sent Events) pour les mises à jour temps réel de la grille planning
❌Pas de dark mode · Pas de SMS · Interface français uniquement
